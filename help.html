DDLC 中文 Mod 模板帮助

声明

新版 DDLC 中文 Mod 模板是 Doki Doki Literature Club 的中文 Mod 制作模板，与 Team Salvato 无关。您应当在通关原游戏后使用此 Mod 模板。Mod 需要 DDLC 原版文件，您可以在 DDLC.moe 或者 Steam 下载。

安装

模板会修改 DDLC，因此您必须获取原版 DDLC。您可以在 DDLC.moe 或者 Steam 下载。

您可以将本 ZIP 压缩包内的所有文件解压并复制到 DDLC 的游戏文件夹内，如果有提示，请允许覆盖所有文件。接下来，打开游戏，你将看到修改后的 DDLC，此时 Mod 就安装完成了。

Windows（Steam 版本）

如果您的 DDLC 是从 Steam 获取的，请在 Steam 库中右键单击 DDLC，选择 管理 → 浏览本地文件，这样就可以查看 DDLC 的本地文件并进行 Mod 安装了。

macOS

Mac 用户可以通过右键单击 DDLC.app 文件，选择 显示包内容，然后转到 Contents/Resources/autorun 文件夹，这样就可以查看 DDLC 的本地文件并进行 Mod 安装了。

卸载 Mod

如果您想还原原版 DDLC，您可以手动删除 Mod 模板的所有文件，或者重新在 DDLC.moe 或者 Steam 下载 DDLC。

特殊情况（对于使用 Ren'Py 7 构建的 Mod）

某些情况下，Mod 压缩包内可能会包含 .exe 等类的可执行文件。由于 Ren'Py 兼容性问题，请您仍然将 Mod 压缩包内的文件解压到 DDLC 文件夹内，但请使用压缩包附带的可执行文件启动游戏。

基本说明

要进行游戏，请使用鼠标左键或空格、回车键阅读文本。遇到选项时，请使用鼠标指针指向并左键选择，或使用方向键选择一个选项，按下回车键选择。

游戏菜单

游戏过程中，按下鼠标右键或 Esc 键，会出现如下菜单：

返回
返回游戏。
存档
允许您通过点击来保存游戏进度到指定的存档位中。
读档
允许您通过点击指定存档位中的存档加载游戏进度。点击「自动」将会访问自动存档的存档位（如果 Mod 提供）。
设置
改变游戏的偏好设定（显示 / 阅读）：
显示模式
在全屏幕与窗口模式之间切换。
文字显示速度
控制每一行文本的显示速率，滑块越往右则越快。当滑块最靠右时，每行文本将瞬间显示完毕。
快进设置 - 未浏览过的内容
Chooses between skipping messages that have been already seen (in any play through the game), and skipping all messages.
快进设置 - 选择选项之后
Controls if skipping stops upon reaching a menu.
自动翻页时间
Controls automatic advance. The further to the left this slider is, the shorter the amount of time before the game advances. All the way to the right means text will never auto-forward.
Music and Sound Volume
Controls the volume of the Music and Sound effect channels, respectively. The further to the right these are, the louder the volume.
主菜单
Returns to the main menu, ending the current game.
退出
Exits the game; the game will be closed and ended.

Key and Mouse Bindings

Left-click, Enter
Advances through the game, activates menu choices, buttons, and sliders.
Space
Advances through the game, but does not activate choices.
Arrow Keys
Navigates between menu choices, buttons, and sliders.
Ctrl
Causes skipping to occur while the ctrl key is held down.
Escape
Enters the game menu. When in the game menu, returns to the game.
Right-click, H
Hides the text window and other transient displays.
F
Toggles fullscreen mode
S
Takes a screenshot, saving it in a file named screenshotxxxx.png, where xxxx is a serial number.
Alt-M, Command-H
Hides (iconifies) the window.
Alt-F4, Command-Q
Quits the game.
Delete
When a save slot is selected, deletes that save slot.

Loading and Deleting Save Data

Some mods may offer the option to load some or all of your data from a previous mod playthrough. If so, that option will be presented on the first load of the game, after the initial content warning and disclaimer.

If you do choose to load previous data, the original mod save data will not be changed.

In some cases, you may want to delete your save data to start the game completely over. To do this, navigate to the %APPDATA%/RenPy folder, double-click the folder of the mod and delete the files located in it and launch DDLC again.

Deleting save data for a mod will not alter or delete original save data from DDLC.
